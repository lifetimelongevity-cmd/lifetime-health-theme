# Documentation Archive

Historische oder superseded Dokumente werden hier aufbewahrt, damit Kontext erhalten bleibt, ohne die aktive Guidance zu verwässern.

## Structure

| Folder | Use |
|---|---|
| `prompts/` | one-off implementation briefs and agent prompts |
| `drafts/` | superseded blueprints or earlier spec versions |
| `legacy-root-files/` | formerly root-level notes that are not active project guidance |
| `pdp-drafts/` | old product-template blueprints and copy packs |

## Rules

- Nichts hier ist Source of Truth.
- Inhalte nur zurück in `docs/` holen, wenn sie bewusst reaktiviert und aktualisiert werden.
- Code-/asset-bezogene Archive bleiben weiterhin unter `/_archive/`.
